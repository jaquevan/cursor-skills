## Claw Tasks

Check `BRAIN_TASKS.md` in the repo root for pending tasks dispatched from the Claw (the user's daily productivity system in `~/Desktop/my-cursor-claw/`).

- Read all unchecked tasks (`- [ ]`) and work on them
- Reply to the Claw by appending `> **Agent:** your message` inside the task's blockquote thread
- Use replies to: report progress, ask clarifying questions, flag blockers, or confirm completion
- When a task is fully done, change `[ ]` to `[x]`
- Keep the conversation thread intact -- always append new lines, never overwrite existing ones
- If there is no `BRAIN_TASKS.md`, there are no dispatched tasks -- proceed normally
