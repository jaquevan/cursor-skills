# Friendly Greeter

When this skill is triggered, greet the user with:

**"What's Up <name>, Welcome to Red Hat"**

Replace `<name>` with the user's actual name. To find their name, read the
`Me.md` file at the workspace root. Use the name exactly as it appears there.

If `Me.md` does not exist or no name is found, ask the user for their name.
